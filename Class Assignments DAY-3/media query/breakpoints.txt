# Positions:

# JAVASCRIPT
-> high level language: garbage collector
-> dynamically typed language: run time
-> Loosely/weakly typed: syntaxing is loose.
-> synchronous: line by line as we have written
-> single threaded
-> blocking in nature: once a task is completed it cant go to another one.
-> async -> helper thread -> still singly threaded -> try to execute things faster but at them comes to the main thread.


# Working of JAVASCRIPT:
-> render engine and JS engine:
-> types of JS engine: 
     1. V8 - present inside DOM, chrome, node req., deno, edge.....
     2. Spider monkey - used inside firefox
     3. JavaScript core - used in safari, bun....
     4. Chakra - used in internet explorer

JS File -> Parser/lexer (component) - syntax error and tokenisation/lexical analysis -> AST (abstract syntax tree) -> interpreter -> Profiler (repeating lines) hot/warm code -> JIT compiler (just in time) -> interpreter.
 
-> types of environment:
     1. Browser - run time environment
     2. Non-Browser - run time environment -> run at Node, Bun, Deno.......

Mocha -> LiveScript -> JavaScript  
ECMAScript -> ES -> ES6 (2015) -> current version of JS is ES2025
